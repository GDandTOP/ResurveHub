## 반응형 디자인

### 데스크톱 (md 이상)
- 사이드바 고정 표시 (왼쪽 264px)
- 헤더 상단 고정
- 메인 컨텐츠 사이드바 우측

### 모바일 (md 미만)
- 사이드바 숨김 (햄버거 메뉴로 토글)
- 오버레이로 표시
- 전체 화면 컨텐츠

## 색상 체계

### 기본 색상
- **Primary**: 검은색 계열 (`oklch(0.28 0 0)`)
- **Accent**: 밝은 회색 (`oklch(0.95 0 0)`)
- **Muted**: 중간 회색 (`oklch(0.5 0 0)`)
- **Border**: 연한 회색 (`oklch(0.90 0 0)`)

### 상태 색상
- **Success**: 초록색 (#10b981)
- **Warning**: 주황색 (#f59e0b)
- **Danger**: 빨간색 (#ef4444)

## 문제 해결

### 개발 서버가 실행되지 않을 때
```bash
# 포트가 사용중인 경우
lsof -ti:3000 | xargs kill -9

# 다시 실행
npm run dev
```

### 빌드 오류 발생 시
```bash
# 캐시 삭제
rm -rf .next

# 재설치
npm install

# 다시 빌드
npm run build
```

### 타입 오류 발생 시
```bash
# TypeScript 체크
npx tsc --noEmit
```
